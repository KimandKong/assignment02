### 문제1


![과제1](https://user-images.githubusercontent.com/93643813/144736162-f3bba1ad-06e3-4e6a-8255-60be99004ec8.gif)


* 최고 스코어 __8 points__ 

* 사용한 명령어 : GWi"<End>"<Esc>ZZ

| 명령어 | 사용이유 |
|:---|:---|
| G |파일의 마지막 줄로 이동 하는 것 입니다.  |
| W | 문자단위 이동하는 것입니다.( 소문자w는 다음단어의 처음으로 이동할수 있습니다.) |
| i | 현위치에서 입력모드를 시작할수 있습니다. |
| ZZ | 저장하고 나가기 키를 써서 나갔습니다.(:wq와 같은 기능입니다.) |


* 설명

1.처음에는 G를 입력해서 파일의 마지막으로 이동합니다.

2.W를 둘러 문자단위로 이동합니다.

3.i로 현위치에서 입력모드를 하여서 __"__ 를 입력하고 End키를 누르고 다시한번더 __"__ 를 입력합니다.

4.완료가 되면 Ese키를 눌러서 일반모드로 돌아오게 합니다. 

5.마지막으로 종료키 ZZ를 눌러 종료를 해줍니다.

### 문제3


![과제3](https://user-images.githubusercontent.com/93643813/144740066-d2ca0c9a-7e7e-4633-a8fe-c207bbd42c69.gif)


* 최고 스코어 __35 points__ 

* 사용한 명령어 : 5<CR>yyPWCTODO<Home><Right>// <Esc>yykPWcwVersion<Esc>ZZ

| 명령어 | 사용이유 |
|:---|:---|
| :<number> | 5번째 줄로 이동하게 하는 것입니다.  |
| yy | 현재 커서가 위치하는 줄을 복사합니다. |
| shif+p | 현재 커서가 위치한 바로 윗 줄에 붙혀넣기를 합니다. |
| W | 문자단위 이동하는 것입니다.( 소문자w는 다음단어의 처음으로 이동할수 있습니다.) |
| cw | 커서가 위치한 단어를 잘라내고 insert를 합니다. |
| C | 커서가 위치한 곳부터 단어의 마지막까지 잘라내고 insert를 합니다. |
| k | 위로 올리기 기능입니다. |
| ZZ | 저장하고 나가기 키를 써서 나갔습니다.(:wq와 같은 기능입니다.) |



* 설명

1.처음에는 :5를 입력해서 그 해당 줄로 이동합니다.

2.yy를 눌러 그 줄을 복사합니다.

3.복사가 완료 되었으면, shift+p를 눌러서 위에 줄에 붙혀넣기를 합니다.

4.W를 눌러서 Bool의 위치에 있게 합니다.
 
5.커서가 Bool의 위치에 있으면 C를 눌러서 그 단어를 잘라내고 insert를 "TODO"로 합니다.

6.HOME키를 누르고 다시 오른쪽 버튼을 누른후 "// "룰 눌러주고 esc키를 일반모드로 돌아오게 합니다.
  
7.다시 yy를 눌러서 복사를 합니다.
  
8.k를 눌러 위로 올라간다음에, shift+p를 눌러 위에 줄에 붙혀넣기를 합니다.

9.W로 Debug의 위치에 가게 한다음 cw를 누른후 , "Version"을 입력하고 다시 esc키를 눌러 일반모드로 돌아오게 합니다.
  
10.ZZ를 눌러서 종료합니다.

11.마지막으로 종료키 ZZ를 눌러 종료를 해줍니다.

### 문제4




https://user-images.githubusercontent.com/93643813/144744832-542ade01-5123-4ce4-940c-ad67853864ba.mp4



* 최고 스코어 __60 points__ 

* 사용한 명령어 :?k<CR>rgkrrkrb:%s/y1/abs(y1)/g<CR><C-A><C-A><C-A>w<C-A><C-A><C-A>W<C-A><C-A><C-A>:4<CR><C-A><C-A>w<C-A><C-A>W<C-A><C-A>:3<CR><C-A>w<C-A>W<C-A>ZZ

  

| 명령어 | 사용이유 |
|:---|:---|
| ?<찾고싶은단어> | 위쪽으로 검색을 합니다.  |
| %s/<현재문자>/<바꿀문자>/g |현재 문자를 입력하고 바꿀문자를 입력하면 현재문자가 바꿀문자가 됩니다. 여기서g는 글로벌로 전체를 의미합니다.  |
| Ctrl + a | 숫자가 1단위로 커집니다. |
| r | 문자를 바꿀수 있습니다. |
| W | 문자단위 이동하는 것입니다.( 소문자w는 다음단어의 처음으로 이동할수 있습니다. |
| ZZ | 저장하고 나가기 키를 써서 나갔습니다.(:wq와 같은 기능입니다.) |



* 설명

1. 처음은 k를 ?로 찾습니다. 즉, 위쪽으로 부터 검색을 합니다. 

2. r을 눌러서 "g,r,b"순으로 바꾸어 줍니다.

3. %s/<현재문자>/<바꿀문자>/g 를 사용해서 "y1"을 "abs(y1)"으로 변경 합니다.

4. 위와 같이 변경되면 커서가 맨아래줄을 가르키고 있는데 여기서, ctrl + a를 누르면 숫자쪽으로 향하고 숫자가 1식 증가합니다.
 
5. 변경이 완료가 되면 w를 눌러서 조금식 이동하고 마지막에는 W를 눌러서 #이 있는 부분까지가서 ctrl + a를 클릭해서 문제에서 요구하는 만큼 변경합니다.
  
6. 완료가 되면 :<행> 을 눌러 그다음행에서 4,5번의 행동을 반복하면 됩니다.
  
7. 완료가 되면 ZZ를 눌러 종료를 합니다.

### 문제5

![과제5](https://user-images.githubusercontent.com/93643813/144747872-c9fbcf0c-2d74-41b7-9e1b-a54c4066b772.gif)


* 최고 스코어 __31 points__ 

* 사용한 명령어 : 5<CR>yw:10<CR>f"pa,name,age,score<Esc>ZZ

| 명령어 | 사용이유 |
|:---|:---|
| :<number> | 5번째 줄로 이동하게 하는 것입니다.  |
| f<char> | 내가 찾고 싶은 char를 누르면 시작하는 단어로 이동합니다.|
| a | 현재 위치 다음 칸에서insert를 시작합니다.|
| p | 저장한것을 커서위치에서 부터 붙혀넣기 합니다.  |
| yw | 커서에서 있는 위치부터 한단어를 저장합니다. |
| ZZ | 저장하고 나가기 키를 써서 나갔습니다.(:wq와 같은 기능입니다.) |




* 설명

1. 처음에는 :5를 눌러 5번째 줄로 이동합니다.

2. yw눌러서 커서에서 있는 단어를 복사했습니다.

3. :10 을 눌러서 그 해당 장소로 가고 f"를 해주어서 "가 있는 단어 앞으로 갔습니다.

4. p를 눌러서 붙혀넣기를 하고 원하는 답을 넣어줍니다.
 
5. 마지막으로 종료키 ZZ를 눌러 종료를 해줍니다.
